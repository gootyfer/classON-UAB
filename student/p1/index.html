Introducción

De la aplicación de este modelo en el contexto educativo de este caso se define un entorno tecnológico para dar apoyo a la puesta en marcha de la experiencia:

  • Para estructurar las distintas fases de la experiencia y avisar a los alumnos de la actividad que debe realizar se usa el sistema Gestor de Cursos Moodle. Moodle incoropora una funcionalidad para la entrega de proyectos y realización de cuestionarios que se usan para facilitar la entrega de las presentaciones a los alumnos y el cuestionarios de la fase final, respectivamente.
  • Para el cuestionario inicial se usa Google Docs Spreadsheets.
  • Para la experiencia de exploración del campus a pie se usan 50 tarjetas NFC y 12 móviles NOKIA (N6131, N6212) con lector de tarjetas NFC.
  • Para estructurar la organización de tarjetas a través del campus se usa una herramienta que permite diseñar es espacio físico asociado a las distintas fases de una actividad 4SPPIces_space
